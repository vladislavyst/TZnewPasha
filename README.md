# Интерактивный тест для сбора ТЗ

Веб-приложение для сбора технического задания от клиентов по разработке сайта. Клиент проходит интерактивный тест из 76 вопросов по 9 разделам, выбирает варианты ответов и добавляет комментарии. В конце автоматически генерируется полное ТЗ, которое можно скачать или отправить на email.

## Возможности

- ✅ 76 вопросов по 9 основным разделам
- ✅ Выбор одного или нескольких вариантов ответа
- ✅ Произвольные комментарии к каждому вопросу
- ✅ Автоматическая генерация ТЗ в формате Word (.docx)
- ✅ Автоматическое скачивание Word документа при завершении теста
- ✅ Отправка готового ТЗ на email
- ✅ Скачивание ТЗ в текстовом формате
- ✅ Адаптивный дизайн для всех устройств
- ✅ Индикатор прогресса прохождения теста

## Технологии

- **Next.js 14** - React фреймворк с App Router
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Nodemailer** - отправка email
- **docx** - генерация Word документов
- **file-saver** - скачивание файлов

## Установка

1. Клонируйте репозиторий или скачайте файлы
2. Установите зависимости:

```bash
npm install
```

**Важно:** Если возникают проблемы с установкой из-за кодировки PowerShell, используйте один из способов:
- Откройте обычную командную строку (cmd) и выполните `npm install`
- Дважды кликните на файл `install.bat` в корне проекта
- Откройте терминал в VS Code и выполните `npm install`

3. Настройте переменные окружения:

Создайте файл `.env.local` на основе `.env.example`:

```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=your-app-password
SMTP_FROM=your-email@gmail.com
```

**Важно:** Для Gmail нужно использовать [App Password](https://support.google.com/accounts/answer/185833), а не обычный пароль.

4. Запустите dev сервер:

```bash
npm run dev
```

5. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Разделы теста

1. **Контент и медиа** (10 вопросов) - логотипы, тексты, фото, видео
2. **Структура каталога** (10 вопросов) - количество товаров, категории, фильтры
3. **Функционал форм** (8 вопросов) - типы форм, валидация, обработка
4. **Интеграции** (9 вопросов) - CRM, 1С, Telegram, аналитика
5. **Административная панель** (8 вопросов) - права доступа, функции админки
6. **Дизайн и брендинг** (10 вопросов) - цвета, стиль, анимации
7. **Технические требования** (8 вопросов) - домен, хостинг, SEO
8. **Сроки и бюджет** (7 вопросов) - дедлайны, оплата, поддержка
9. **Юридические аспекты** (6 вопросов) - документы, политика конфиденциальности

## Настройка SMTP

### Gmail

1. Включите двухфакторную аутентификацию
2. Создайте App Password: [Google Account Settings](https://myaccount.google.com/apppasswords)
3. Используйте этот пароль в `.env.local`

### Другие провайдеры

**Yandex:**
```env
SMTP_HOST=smtp.yandex.ru
SMTP_PORT=465
SMTP_USER=your-email@yandex.ru
SMTP_PASSWORD=your-password
```

**Mail.ru:**
```env
SMTP_HOST=smtp.mail.ru
SMTP_PORT=465
SMTP_USER=your-email@mail.ru
SMTP_PASSWORD=your-password
```
