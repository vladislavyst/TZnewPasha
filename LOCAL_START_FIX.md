# Исправление проблемы с локальным запуском

## Проблема

Проект не запускался локально, потому что **отсутствовала обязательная папка `app/`** с файлами для Next.js 14 App Router.

## Что было исправлено

✅ Создана минимальная структура проекта:

1. **`app/layout.tsx`** - корневой layout компонент (обязателен для App Router)
2. **`app/page.tsx`** - главная страница (обязательна для App Router)
3. **`app/globals.css`** - глобальные стили с Tailwind CSS

## Теперь проект должен запускаться

Попробуйте запустить:

```bash
npm run dev
```

Проект должен запуститься на http://localhost:3000

## ⚠️ Важно

Создана только **минимальная структура** для запуска. Для полноценной работы приложения нужно восстановить:

- ❌ `app/test/page.tsx` - страница прохождения теста
- ❌ `app/results/page.tsx` - страница результатов
- ❌ `app/api/send-tz/route.ts` - API для отправки email
- ❌ `components/` - все компоненты React
- ❌ `lib/` - утилиты, типы и вопросы
